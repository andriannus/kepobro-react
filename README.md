# KepoBro - React.js

> NewsAPI with React.js

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm start

# run test runner
$ npm test

# build for production and launch server
$ npm run build
```
